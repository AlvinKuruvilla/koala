§ 00

A fast HTML renderer, from scratch.

A spec-driven web engine written in Rust, paired with a native Qt chrome modeled on Ladybird. Built to read like its own specification.

§ 01   Engine
01
CSS 2.1

Layout, from the spec up.

Block, inline, flexbox, grid, and table formatting contexts implemented straight from CSS 2.1 and the Box Model Level 3 drafts.

02
WHATWG HTML

Tree construction tracking the living standard.

A tokenizer and tree builder that match the WHATWG HTML parsing algorithm, including all 2,231 named character references.

03
Pure Rust

No C++ on the engine side.

Rendering pipeline, text shaping, and image decoding all written in Rust. The Qt chrome sits across a single cxx bridge.
